# Writing Assistant Frontend

## 项目概述

这是一个基于人工智能的写作助手前端项目，旨在帮助用户提高写作效率和质量。该项目集成了 AI 聊天、文本编辑、知识库管理等功能，为用户提供全方位的写作支持。

## 核心功能

1. AI 辅助写作

   - AI 聊天交互
   - 智能写作建议
   - 文本生成

2. 编辑器功能

   - 富文本编辑
   - 文档大纲管理
   - 版本历史控制

3. 知识库系统

   - 个人知识库
   - 系统知识库
   - 文件管理

4. 用户系统
   - 用户认证
   - 个人中心
   - 历史记录

## 项目目标

1. 提供直观易用的写作界面
2. 实现高效的 AI 辅助写作流程
3. 建立完整的知识管理体系
4. 确保良好的用户体验和系统性能

## 技术栈

- React + UmiJS + TypeScript
- Ant Design 组件库
- Less CSS 预处理器
- RESTful API 接口

## 项目规范

- 遵循 TypeScript 类型规范
- 使用 ESLint 进行代码检查
- 采用 Prettier 进行代码格式化
- 遵循组件化开发原则
